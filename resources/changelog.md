## v3.2.0 - 2026-05-15

**崩溃修复：**
- 修复侧边栏打开后程序崩溃：删除 QGraphicsOpacityEffect 透明度动画，避免 use-after-free 问题
- 修复页面切换时潜在的崩溃风险：移除页面淡入动画中的 QGraphicsOpacityEffect race condition
- 修复 exe 属性描述乱码：资源文件改用 ASCII 编码描述文字

**功能改进：**
- 单实例运行：同时启动两个实例时，第二个实例自动退出，不创建新窗口
- 侧边栏新增自动收回选项：设置页可勾选"导航后自动收回侧边栏"，默认开启
- 新增亚克力效果开关：设置页可启用 Win11 亚克力风格（半透明背景）
- 首次启动新增视觉效果设置页：可配置亚克力效果和侧边栏自动收回

**视觉优化：**
- 侧边栏动画升级为 OutBack/InBack 缓动曲线，动画更生动
- 侧边栏动画时长调整为 350ms（打开）/ 300ms（关闭）
- 主题按钮增大尺寸并添加 padding，防止文字被遮盖
- 首次启动上一步按钮改为自适应宽度并添加 padding，防止文字被遮盖
- 主界面标题从"cn国防装备"改为"中国国防装备"
- 删除无用的字体大小调节功能

**性能优化：**
- 优化侧边栏动画：复用动画对象，避免每次创建新对象导致内存泄漏
- 优化答题模块信号连接：使用 Qt::UniqueConnection 避免重复连接/断开信号
- 移除答题完成后不必要的信号重连逻辑
- 移除 Windows 原生亚克力 API 调用（导致标题栏消失），改用 Qt 半透明背景

**工具脚本：**
- 新增 build.bat 一键编译部署脚本
